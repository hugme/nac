Thruk Servers

The Thruk servers are the controling nodes for Nagios. The primary are redundant, if you change one the other will change. They control the main nagios servers which normally are the ones who page out. The second set of servers control the backups. They are exactly the same as the first set except their mail is dropped to /dev/null.
